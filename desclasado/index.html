Desclasado CSS


Un archivo de estilos CSS para diseñar facilmente sitios web con HTML de forma semantica sin agregar clases

Cómo usar DESCLASADO CSSDescargar

Descalsado CSS es un único archivo de estilos CSS que funciona que itenta brindar una forma simple y liviana para poder diseñar sitios web con HTML semántico sin la necesidad de agregar clases. HTML semántico significa que el archivo HTML puede usar unicamente etiquetas HTML según el nombre de cada una, sin la necesidad de agregar nombres de clases, estilos en línea, etiquetas div o span.

Desclasado CSS está configurado en dos versiones, según la preferencia del usuarie: FLEX y GRID. Ambas versiones son idénticas, salvo que usan FLEX o GRID para crear columnas y tarjetas (cards).

También está diseñado con dos temas CLARO y OSCURO. Estos temas si pueden elegir fijos o para que cambien de forma automática detectando el tema que utiliza el navegador de quién visite la pagina web. ¡El esquema de colores de estos temas es muy fácil de personalizar!

Desclasado CSS está basado en MVP.CSS, Basic.CSS, Simple.css, Classless.css y d0
