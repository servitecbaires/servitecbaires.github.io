Documentación


Desclasado CSS es muy fácil de usar. Solo necesitas saber como escribir una página web en HTML. Ni siquiera es necesario tener conocimientos de CSS

Cada etiqueta HTML cumple una función especifica según su nombre lo indica. Puedes utilizar Desclasado CSS para que tu página luzca elegante y lista para publicar en la web en segundos y sin la necesidad de crear .clases, ajustes adicionales de CSS ni necesitas llenar tu código de etiquetas <div> o <span>

En esta página se muestra la lista de etiquetas HTML que uitiliza Desclasado CSS y un ejemplo de su aspecto en el navegador. Toda esta página esta utilizando Desclasado CSS    ; )

Existen dos versiones de Desclasado CSS para realizar un esquema de columnas o tarjetas.

  • Formatos de texto


    Texto en negritas - Texto en <b>negritas</b> o Texto en <strong>negritas</strong>
    Texto en itálicas - Texto en <i>itálicas</i> o Texto en <em>itálicas</em>
    Texto subrayado - Texto <u>subrayado</u>
    Texto resaltado - Texto <mark>resaltado</mark>
    Texto tachado - Texto <s>tachado</s>
    Texto superíndice - Texto <sup>superíndice</sup>
    Texto subíndice - Texto <sub>subíndice</sub>
    Texto con código de programación - Texto con <code>código de programación</code>
    Texto con indicación de teclas del teclado Alt+F4 - Texto con indicación de teclas del teclado <kbd>Alt+F4</kbd>
    Texto con etiqueta o señalador - Texto con etiqueta <sup><mark>Alt+F4</sup></mark>

Imágenes

  • Imagen alineada con el texto <p><img> ... </p>

    Una imagen dentro de una etiqueta de párrafo <p> se alineará automáticamente a la derecha. Esto es útil (sobre todo) cuando la imagen es pequeña. Proin posuere varius justo, id sagittis quam lobortis sed. Integer sed erat neque. In tristique ut ipsum et faucibus. Nulla a orci commodo, semper velit sed, iaculis ante. Nam feugiat neque nisl, eu pellentesque ipsum fermentum nec. Praesent nec quam vitae dui euismod condimentum.

  • Imagen centrada

    Las imágenes insertadas dentro de las estiquetas <figure> serán centradas automáticamente (en el caso que sean más angostas que la pantalla). Se puede agregar una referencia de texto debajo de la imagen itulizando una etiqueta <figcaption>.
